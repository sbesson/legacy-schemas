Introduction

This document outlines the XML OME Schemas created by the Open Microscopy Environment Group. This is the main schema which defines the OME ontology for microscopy. This schema makes use of the others. Detailed information on the schemas and future development plans are available at http://www.openmicroscopy.org/site/support/file-formats. Further information on the groups work is available at http://www.openmicroscopy.org/

History

This lists the current and prior versions of the OME schema. The schemas are in a process of constant development to extend and refine the OME Data Model.

  • June 2013 - namespace /2013-06/
  • June 2012 - namespace /2012-06/

  • 2003 (FC) - namespace /FC/ome.xsd

Information regarding schemas released before June 2007 is available at http://www.openmicroscopy.org/XMLschemas/. This includes the schemas used by the original OME server.
