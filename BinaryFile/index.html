Introduction

This document outlines the XML BinaryFile Schemas created by the Open Microscopy Environment Group. This is the Binary File schema used to describe a file location, or the location of a fragment within a file. Detailed information on the schemas and future development plans are available at http://www.openmicroscopy.org/site/support/file-formats. Further information on the groups work is available at http://www.openmicroscopy.org/

History

This lists the current and prior versions of the BinaryFile schema. The schemas are in a process of constant development to extend and refine the OME Data Model.

  • January 2015 - namespace /2015-01/
  • June 2013 - namespace /2013-06/
  • June 2012 - namespace /2012-06/
  • June 2011 - namespace /2011-06/
  • June 2010 - namespace /2010-06/
  • April 2010 - namespace /2010-04/
  • September 2009 - namespace /2009-09/
  • September 2008 - namespace /2008-09/
  • February 2008 - namespace /2008-02/
  • June 2007 - namespace /2007-06/
  • 2003 (RC1) - namespace /RC1/BinaryFile.xsd

Information regarding schemas released before June 2007 is available at http://www.openmicroscopy.org/XMLschemas/. This includes the schemas used by the original OME server.
