Introduction

This document outlines the XML MLI Schemas created by the Open Microscopy Environment Group. This is the Matlab Interface sub-schema. Detailed information on the schemas and future development plans are available at http://www.openmicroscopy.org/site/support/file-formats. Further information on the groups work is available at http://www.openmicroscopy.org/

History

This lists the current and prior versions of the MLI schema. The schemas are in a process of constant development to extend and refine the OME Data Model.

  • September 2009 - namespace /2009-09/
  • September 2008 - namespace /2008-09/
  • February 2008 - namespace /2008-02/
  • June 2007 - namespace /2007-06/
  • 2003 (IR2) - namespace /IR2/MLI.xsd

Information regarding schemas released before June 2007 is available at http://www.openmicroscopy.org/XMLschemas/. This includes the schemas used by the original OME server.
